# Zomato Restaurant Ratings Analysis

## Project Overview
This project focuses on analyzing Zomato restaurant data to understand the factors that influence restaurant ratings and customer satisfaction. The analysis is performed using Python for data cleaning and exploration, and Power BI for creating an interactive dashboard.

---

##  Objectives
- Analyze restaurant ratings and pricing trends  
- Study the impact of online delivery on ratings  
- Perform location-based analysis of restaurants  
- Identify patterns affecting customer satisfaction  

---

##  Dataset
- Source: Kaggle – Zomato Bangalore Restaurants Dataset  
- The original dataset exceeds GitHub’s 25MB upload limit.
- A cleaned and reduced dataset used for analysis is included.

 Kaggle Dataset Link:  
https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants
